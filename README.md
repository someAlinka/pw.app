# Parrot Wings Application

<br>
Учтены все требования из https://docs.google.com/document/d/1NMfvzJq173mqQwrMq5-s5PqQ-c6kZE1ZqhTbruGsv80/edit.

Папка node_modules не залита, все в package.json
